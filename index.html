DictMySQL

A MySQL class for more convenient database manipulation with Python dictionary.

View on GitHub Download .zip Download .tar.gz

Initialization

Parameters:

  • host: string
  • user: string
  • passwd: string
  • db: string
  • port: int, default 3306
  • charset: string, default 'utf8'
  • init_command: string, default 'SET NAMES UTF8'
  • dictcursor: bool, default False
    When it's True, the connector uses DictCursor instead of regular cursor, so that all the return of SQL query will be wrapped in dict.
  • use_unicode: bool, default True

Returns: DictMySQL object

Example:

from dictmysql import DictMySQL
mysql = DictMySQL(db='occupation', host='127.0.0.1', user='root', passwd='')

SELECT

Parameters:

  • table: string
  • columns: list, default None
  • join: dict, default None
    See JOIN Syntax for more details.
  • where: dict, default None
    See WHERE Syntax for more details.
  • order: string, default None
  • limit: int|list, default None

Returns: the result of SQL query.

Examples:

mysql.select(table='jobs', where={'value': 'Artist'})
# SELECT * FROM `jobs` WHERE (`value` = "Artist");

mysql.select(table='jobs', 
             columns=['jobs.id', 'profiles.id'], 
             join={'profile_jobs': {'jobs.id': 'profile_jobs.job_id'}, 
                   'profiles': {'profile_jobs.profile_id': 'profiles.id'}}, 
             where={'jobs.value': 'Artist'})
# SELECT `jobs`.`id`, `profiles`.`id` FROM `jobs` 
# JOIN `profiles` ON `profile_jobs`.`profile_id`=`profiles`.`id` 
# JOIN `profile_jobs` ON `jobs`.`id`=`profile_jobs`.`job_id` 
# WHERE (`jobs`.`value` = "Artist");
